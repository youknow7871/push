Push Consent UI - Versions Overview

요청하신 앱 푸시 동의 수집 화면의 3가지 버전을 한눈에 비교할 수 있는 대시보드입니다.
각 버전별 특징적인 인터랙션을 직접 클릭하여 테스트해보세요!

버전 A (Version A)

기본 요구사항 구현
모든 토글이 명시적으로 노출되며 개별 조작 중심

버전 B (Version B)

마케팅 동의 끄기 방지 & 자동 켜짐 연동
알림 설정에 마케팅 토글 노출, 하단 팝업 유도 로직 적용

버전 C (Version C)

Dynamic 팝업으로 사용자 경험 개선
광고 알림 ON 시 자연스러운 팝업으로 마케팅 유도 유도
